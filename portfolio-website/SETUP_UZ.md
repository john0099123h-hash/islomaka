# 🚀 Portfolio Saytni Tezkor O'rnatish

## 1️⃣ Fayllarni yuklab oling

```bash
git clone https://github.com/john0099123h-hash/islomaka.git
cd islomaka/portfolio-website
```

## 2️⃣ index.html ni tahrirlang

### Ismingizni o'zgartiring:
```html
<h1>Hi, I'm <span class="gradient-text">SIZNING ISMINGIZ</span></h1>
```

### Email va telefon:
```html
<strong>sizning.email@example.com</strong>
<strong>+998 XX XXX XX XX</strong>
```

### Social media:
```html
<a href="https://github.com/sizning-username">
<a href="https://t.me/sizning-username">
<a href="https://linkedin.com/in/sizning-username">
```

## 3️⃣ Rasmlarni qo'shing

`images/` papkasiga rasmlarni qo'ying:
- `profile.jpg` - Sizning rasmingiz
- `about.jpg` - About section
- `project1.jpg` to `project6.jpg` - Loyihalar

**Rasm yo'qmi?** Vaqtincha placeholder ishlatish:

`index.html` da rasmlarni almashtiring:
```html
<img src="https://via.placeholder.com/800x800/6366f1/ffffff?text=Profile" alt="Profile">
```

## 4️⃣ Ranglarni sozlang (ixtiyoriy)

`css/style.css` faylida:
```css
:root {
    --primary-color: #6366f1;     /* O'zgartiring */
    --secondary-color: #8b5cf6;   /* O'zgartiring */
    --accent-color: #ec4899;      /* O'zgartiring */
}
```

## 5️⃣ Brauzerda ochish

Oddiy `index.html` ni ikki marta bosing yoki Live Server ishlatib ko'ring.

## 6️⃣ Onlayn joylashtirish

### GitHub Pages (BEPUL):
1. GitHub'da yangi repo yarating
2. Fayllarni yuklang
3. Settings → Pages → Source: main
4. Tayyor! Link oling

### Netlify (BEPUL):
1. [netlify.com](https://netlify.com) ga kiring
2. Papkani drag & drop qiling
3. Tayyor!
